# End-to-End Retail Analytics Pipeline: Synthetic Data & Customer Segmentation

An end-to-end data science framework showcasing localized retail simulation, Exploratory Data Analysis (EDA), RFM customer segmentation, and lifecycle-based inventory markdown strategy optimization.

## 📌 Project Overview
This project simulates a realistic Thai retail environment to solve core business optimization problems. It consists of a complete pipeline that generates synthetic data across multiple operational tables, tracks customer behaviors, segments users via behavioral analytics, and optimizes gross margin recovery for expiring products.

### Key Features
- **Data Synthesis**: Mimics authentic Thai customer profiles, holiday occurrences, and purchase histories.
- **RFM Customer Segmentation**: Groups customers into operational tiers (`Champions`, `Loyal`, `At-Risk`, `Hibernating`) for targeted win-back and loyalty plays.
- **Expiry Markdown Engine**: Automatically builds pricing optimization rules to preserve gross margins on aging batch units.
